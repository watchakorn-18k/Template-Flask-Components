# Template-Flask-Components

Template Flask Concept Components

# Virtualenv

```cmd
pip install virtualenv

virtualenv flask
```

# Use Virtualenv

```
# On windows
flask/Scripts/activate.bat

# On Linux
source flask/bin/activate
```

# Install Flask and Black

```
pip install requirements.txt
```

# Run

```
python app.py
```
